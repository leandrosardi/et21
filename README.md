# ET21

Repositorio de ejercicios y prácticas para la materia de programación de la Escuela Técnica N°21.

## Descripción

Este proyecto contiene los ejercicios resueltos a lo largo del cursado. Cada ejercicio está organizado en su propia carpeta o archivo, con el objetivo de practicar y afianzar los conceptos de programación vistos en clase.

## Contenido

- Ejercicios prácticos por unidad temática
- Código comentado para facilitar la comprensión
- Ejemplos y soluciones propuestos por el docente y desarrollados por los alumnos
